Course Overview · Level 1 Basic Training

From sketch
to slicer-ready.

This course will provide students with a basic understanding of 3D design and printing. They will learn how to design 3D models in CAD software. Students will have the opportunity to build a 3D printer and develop a basic understanding of the machine, machine operation, design elements, and best practices.

The course teaches the foundational knowledge needed to start producing quality 3D models to sell and gain employment in engineering fields.

Class Projects

Four hands-on
projects.

Learn to design and create innovative product prototypes, craft functional and stylish furniture, sculpt detailed humanoid models, and engineer precision automotive parts.

Curriculum · What You'll Learn

Four modules.
End-to-end skill.

From terminology and software through building a printer and exporting your first slice file.

Module 01

General Knowledge

In this course the student will learn overall basic knowledge pertaining to terminology, software, and best practices.

  • Terminology
  • Software (CAD & Slicer)
  • Safety and Best Practice
  • Machine Operation
Module 02

Building Your First 3D Printer

In this course the student will build their own 3D printer from start to finish.

  • Frame Assembly
  • Electrical Components
  • Motors X · Y · Z
  • Heated Bed
  • Arduino (Brain)
  • Control Screen
Module 03

Basic Design for 3D Printing

In this course the student will learn how to design objects in Tinkercad (3D design software).

  • Text
  • Shapes
  • Layering
  • Object Depth
Module 04

Basic Principles of 3D Printing Your Model

In this course students will learn how to setup files for 3D printing.

  • Slicing
  • Machine Calibration
  • Heated Bed vs. Cold Bed printing
  • File formats
  • SVG, OBJ, and STL exporting
